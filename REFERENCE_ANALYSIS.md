### Dashboard Layout
- **Sidebar Navigation** (Left side, always visible):

  - Organized into sections: MAIN, LEARNING, AI TOOLS, OTHER
  - Icons for each menu item
  - Clear visual hierarchy

- **Main Content Area**:
  - Personalized welcome message with user name
  - Stats cards showing: Overall Progress, Topics Covered, Mastered Topics, Avg Test Score
  - Knowledge Map section (visual representation)
  - Recent Assessments list
  - Quick Actions panel (right side) with prominent CTAs
  - Notifications panel

### Design Patterns
1. **Card-based layout** for content organization
2. **Color-coded stats** (blue for progress, green for topics, purple for mastery, orange for scores)
3. **Icon-driven navigation** with clear labels
4. **Prominent Quick Actions** in sidebar/panel
5. **Empty states** with helpful messages ("No learning data yet. Start practicing!")
6. **Professional typography** with clear hierarchy
